RDF that fits how Kotlin apps are built

Domain-first Kotlin: keep business types clean, reach RDF through an explicit side channel—DSL, handles, and optional KSP codegen. RDF 1.2 and SHACL 1.2 are first-class; swap Jena, RDF4J, memory, or SPARQL endpoints with configuration. Kastor is an API layer—not a database; providers do storage and protocol heavy lifting.

What makes Kastor different
