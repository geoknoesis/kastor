# Kastor logo assets

| File | Use |
|------|-----|
| `kastor-logo.png` | Default square mark (README, light UI). |
| `kastor-logo-dark-bg.png` | Dark themes, slide decks, GitHub dark-mode banners. |
| `kastor-logo-icon.png` | Favicons and small avatars (simplified; test at 32–64 px). |
| `kastor-logo-wordmark.png` | Headers, wide banners, “mascot + name” horizontal lockup. |

**Guidelines:** Prefer scaling PNGs proportionally; do not stretch. For crisp favicons, export from `kastor-logo-icon.png` at target sizes (e.g. 32×32, 48×48) in your build or design tool.

All marks are project-specific artwork for the Kastor RDF framework (`com.geoknoesis.kastor`).
